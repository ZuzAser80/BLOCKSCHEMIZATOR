</h2>Установка и запуск</h2>
</p>
Приложение написано на C# и требует .NET Runtime.
</p>
<b>Скачайте и установите .NET Runtime:</b>

    Перейдите на официальный сайт .NET

    Скачайте .NET Runtime (не SDK, если не планируете разработку)

</p>
<B>Проверьте наличие .NET:</b>

</p>

**dotnet --info**

</p>
<b>Запуск приложения </b></p>
Если у вас есть .exe файл

    Откройте терминал в папке с приложением

    Выполните:    

    имя_программы.exe

Если у вас есть .dll файл

    Откройте терминал в папке с приложением

    Используйте:
    
    dotnet имя_программы.dll
